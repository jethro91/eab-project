Berikut merupakan berbagai cara Absensi Konvensional dan berbagai masalahnya. Berdasarkan berbagai masalah dibawah ini, maka kami menciptakan Eabsen sebagai solusi absensi Online Realtime Berbasis Cloud.

Absen Manual

  • Harus ada petugas yang dipercaya & harus datang lebih cepat.
  • Kemungkinan yang melakukan absen orang lain
  • Laporan harus diambil & dikirim secara fisik
  • Rekapitulasi harus dilakukan manual satu per satu
  • Memakan banyak waktu
  • Kemungkinan hilang kertas absen
  • Tidak memungkinkan untuk mencari data lama seiring dengan laporan yang betambah banyak
  • Data kehadiran tidak bisa lihat secara langsung (real time)

Absen Sidik Jari Offline

Absen Sidik Jari Offline

  • Laporan harus dikirimkan secara manual dari masing-masing lokasi berbeda
  • Laporan rawan dimanipulasi pada saat dikirimkan
  • Harus ada operator yang bisa dipercaya ditiap lokasi untuk membuat laporan setiap hari.
  • Untuk mencari data lama sangat sulit karena data berada di masing-masing lokasi yang berbeda
  • Data kehadiran ditidak bisa lihat secara langsung (real time) atau harus menunggu laporan dikirim
  • Resiko data hilang ketika mesin absen rusak

Absen Sidik Jari Online Secara Mandiri

  • Harus menyediakan perangkat server dan perangkat pendukung lainya dengan biaya puluhan sampai ratusan juta
  • Harus berlangganan jaringan internet dedicated yang mahal untuk server setiap bulan
  • Instalasi dan setup server/jaringan yang rumit
  • Harus membeli software online dengan biaya yang sangat mahal
  • Harus memiliki orang IT yang siaga 24 jam untuk monitoring, pemeliharaan server (maintenance) dan permasalahan teknis lainnya
  • Harus memiliki backup listrik genset untuk menjamin server hidup selama 24 jam
  • Data-data dapat hilang bila server rusak, terkena bencana alam, permasalahan listrik tidak stabil
  • Harus ada pergantian perangkat server secara berkala untuk peremajaan
  • Apabila adanya penambahan fitur software diperlukan penambahan biaya yang tidak murah
